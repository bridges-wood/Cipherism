# CipherSolve
CipherSolve is a mobile-based classical cipher decryption platform based on statistical analysis 
of their characteristics. 

## Supported Ciphers
1.  Vigenere
2.  Caesar
3.  Substitution.

## Usage
Requries the use of the accompanying Android app.
